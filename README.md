# Countries Explorer Website
## How to Run the Project:
1. Clone the repository.
2. Navigate to the project directory: cd countries-explorer
3. Install dependencies: npm install
4. npm start
   
# This project uses the REST Countries API to fetch country data
1. Get all countries: https://restcountries.com/v3.1/all
2. Search country by name:https://restcountries.com/v3.1/name/{name}
3. Filter countries by region: https://restcountries.com/v3.1/region/{region}
